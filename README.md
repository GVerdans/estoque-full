# Projeto Estoque

Um sistema simples de controle de estoque (Fullstack).

**Backend:** Node + TypeScript + Express + Prisma + JWT

**Frontend:** React + Vite

---

## Status

- Versão backend: v1.0 (concluído em 20 JUL)
- Frontend: iniciado

---

## Conteúdo deste repositório

- `backend/` — API em TypeScript (Express, Prisma, JWT)
- `frontend/` — cliente em React (Vite)

---

Todas as rotas e os detalhes de requisição estão documentados em: [backend/ROUTES.md](backend/ROUTES.md)

## Como rodar (rápido)

1. Instalar dependências (backend)

```bash
cd backend
npm install
```

2. Rodar em modo de desenvolvimento (backend)

```bash
npm run dev
```

3. Frontend (se aplicável)

```bash
cd ../frontend
npm install
npm run dev
```

Observação: ajuste os scripts conforme `package.json` de cada pasta.
